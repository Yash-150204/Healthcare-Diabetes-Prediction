# 🏥 Healthcare Diabetes Prediction

This project uses Machine Learning to predict diabetes risk based on medical data.

---

## 📊 Dataset
Pima Indians Diabetes Dataset

---

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib & Seaborn
- Scikit-learn

---

## 🤖 Model
Logistic Regression

---

## 📈 Features
- Exploratory Data Analysis (EDA)
- Data Visualization
- Model Training & Testing
- Accuracy Evaluation (~74%)
- Confusion Matrix
- Classification Report
- Feature Importance
- User Input Prediction

---

## 🎯 Conclusion
This project demonstrates how machine learning can assist healthcare professionals in early detection of diabetes risk.

---

## 🚀 Future Scope
- Web app (Streamlit)
- Multi-disease prediction
- AI healthcare assistant
